Table

This is how you code a table

Tag Name Description
table Table Tag you need to start a table
thead Table Header The first Row of a table is defined as the header
th Heading row Data with in the heading row
tbody Table Body The Rest is the table body
tr Table Row Each row of the table
td Table Data Each cell of the table
colspan Column Span Defines how many columns a td element should span.
